# cinereview
A sleek, responsive web application for movie reviews, ratings, and recommendations. Built to provide film enthusiasts with a clean UI to explore and track their favorite cinema

Welcome to **The Cine Review**, a sleek, modern, and responsive web application designed for movie enthusiasts. This platform allows users to explore, discover, and read reviews of their favorite movies and TV shows in one place.

 **Live Demo:** [https://thecinereview.netlify.app/]

---

##  Features

* **Discover Movies:** Explore trending, popular, and top-rated movies or TV shows.
* **Detailed Insights:** View comprehensive details including ratings, release dates, genres, and synopsis.
* **Search Functionality:** Quickly find specific movies using an intuitive search bar.
* **Fully Responsive:** Optimized for a seamless experience across desktops, tablets, and mobile devices.
* **Clean UI/UX:** Aesthetic design with dark mode styling, perfect for a cinematic vibe.

---

##  Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript 
* **Styling:**  CSS3
